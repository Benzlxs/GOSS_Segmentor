# GOSS_Segmentor
This repository is for GOSS Segmentor (GST) introduced in the following paper:  
**GOSS: Towards Generalized Open-set Semantic Segmentation**
## Paper and Citation
The paper can be downloaded from [arXiv](https://arxiv.org/abs/2203.12116).
